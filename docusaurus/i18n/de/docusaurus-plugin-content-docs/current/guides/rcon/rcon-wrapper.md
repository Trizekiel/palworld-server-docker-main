# RCON Wrapper

Der GitHub-User [valamidev](https://github.com/valamidev) hat ein Plugin für
`palworld-server-docker` erstellt, das es ermöglicht, den Palworld RCON-Endpunkt mit einem
Bearer-Token zu schützen und über das HTTP-Protokoll darauf zuzugreifen.

Weitere Informationen findest du in seinem GitHub-Repo:
[valamidev/palworld-rcon-buddy](https://github.com/valamidev/palworld-rcon-buddy)
